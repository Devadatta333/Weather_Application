# Weather Application 🌤️

A simple weather application that allows users to search for any city and view real-time weather information.

## Features

* Search weather by city name
* Display current temperature
* Show weather conditions (Sunny, Cloudy, Rainy, etc.)
* Display A-Q-I index quality
* Responsive and user-friendly interface

## Technologies Used

* HTML
* CSS
* JavaScript
* Weather API

## Usage

Enter a city name in the search bar and view the current weather details.

## Future Improvements

* 5-day weather forecast
* Location-based weather
* Dark mode
* Weather icons and animations
